# Cybersecurity Projects Portfolio

This repository contains my cybersecurity academic projects completed as part of my coursework. These projects focus on both theoretical concepts and practical implementation of security techniques.

---

## Minor Project 1  
### Cybersecurity Risk Assessment Framework for Small Businesses

This project focuses on identifying and evaluating cybersecurity risks faced by small and medium enterprises (SMEs).

### Key Work:
- Researched common threats like phishing, ransomware, and insider attacks  
- Developed a risk assessment model based on the NIST framework  
- Created a risk evaluation table using Likelihood × Impact  
- Proposed mitigation strategies such as 2FA, firewalls, and employee training  
- Applied the model through a case study  

---

## Minor Project 2  
### Zero Trust Architecture for Enterprise Security

This project explores the concept of Zero Trust security in enterprise environments.

### Key Work:
- Studied Zero Trust principle: "Never trust, always verify"  
- Implemented authentication models like MFA and RBAC  
- Designed a Zero Trust framework with identity verification and encryption  
- Simulated enterprise security environment  
- Applied access control policies using Identity & Access Management (IAM)  
- Tested security against insider and external threats  

---

## Major Project  
### Phishing Awareness Simulation Using Social Engineering Techniques

This project demonstrates how phishing attacks work and analyzes user behavior.

### Key Work:
- Studied phishing techniques (emails, fake websites)  
- Designed a fake login page using HTML & CSS  
- Hosted project using GitHub Pages  
- Connected frontend to Google Sheets using Google Apps Script  
- Collected real-time user interaction data  
- Analyzed user awareness and response behavior  
- Suggested preventive measures  

---

## Technologies Used

- HTML, CSS  
- GitHub Pages  
- Google Apps Script  
- Google Sheets  
- Cybersecurity Frameworks (NIST, Zero Trust)

---

## Key Learnings

- Users are highly vulnerable to phishing attacks  
- Awareness is the strongest defense  
- Zero Trust improves enterprise security  
- SMEs need structured risk assessment models  
- Simple security practices can reduce major risks  

---

## Preventive Measures

- Use strong passwords  
- Enable Multi-Factor Authentication (MFA)  
- Avoid clicking suspicious links  
- Verify website URLs  
- Regularly update systems  
- Train users about cybersecurity
